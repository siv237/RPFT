# §30. QED 1-loop на $M_4\times K$: вывод формул §4 (gauge fixing, FP, ghost, $\log\det$, $\zeta$)

Этот файл содержит доказательные детали для формул из `00_main.md`, §4 и ссылок из §5.2/§5.6.

## 30.1 Евклидово действие и функциональный интеграл

Рассматриваем евклидову QED на $M_4\times K$:

$$S[A,\Psi]=S_A+S_\Psi+S_{int}$$
$$S_A=\frac{1}{4g_5^2}\int_{M_4}\! d^4x\int_K\! d\mathrm{vol}\; F_{MN}F^{MN},\qquad S_\Psi=\int_{M_4}\! d^4x\int_K\! d\mathrm{vol}\;\bar\Psi\,D\!\!\!\!/\,\Psi.$$

Формальная статистическая сумма:

$$Z=\int\mathcal{D}A\,\mathcal{D}\bar\Psi\,\mathcal{D}\Psi\;e^{-S[A,\Psi]}.$$

## 30.2 Gauge fixing и Фаддеев–Попов для $U(1)$

Фиксируем ковариантную калибровку (Лоренцева):

$$G(A)=\nabla^M A_M=0,\qquad S_{gf}=\frac{1}{2\xi}\int_{M_4}\! d^4x\int_K\! d\mathrm{vol}\; (\nabla^M A_M)^2.$$

Вставка единицы Фаддеева–Попова:

$$1=\Delta_{FP}[A]\int\mathcal{D}\omega\;\delta(G(A^\omega)),\qquad \Delta_{FP}[A]=\det\left(\frac{\delta G(A^\omega)}{\delta\omega}\right).$$

Для $U(1)$ имеем $A^\omega=A+\nabla\omega$, поэтому

$$\frac{\delta G(A^\omega)}{\delta\omega}=\nabla^2,\qquad \Delta_{FP}=\det(\nabla^2).$$

Детерминант удобно представлять через Grassmann ghost-поля $c,\bar c$:

$$S_{gh}=\int_{M_4}\! d^4x\int_K\! d\mathrm{vol}\;\bar c\,\Delta_{gh}\,c,\qquad \Delta_{gh}=-\nabla^2.$$

## 30.3 Гауссовы функциональные интегралы и знаки детерминантов

При 1-loop разложении вокруг фоновой конфигурации сохраняем квадратичную часть по флуктуациям.

Схематически (на уровне гауссиана):

$$\int \mathcal{D}\varphi\;\exp\Big(-\tfrac12\langle \varphi, O\varphi\rangle\Big)\propto (\det O)^{-1/2},$$

откуда вклад в $\Gamma=-\log Z$:

$$\Gamma\supset +\tfrac12\log\det O.$$

Для фермионов:

$$\int \mathcal{D}\bar\psi\,\mathcal{D}\psi\;\exp\big(-\langle\bar\psi, D\psi\rangle\big)\propto \det D,\qquad \Gamma\supset -\log\det D.$$

Для ghost-полей (Grassmann):

$$\int \mathcal{D}\bar c\,\mathcal{D}c\;\exp\big(-\langle\bar c, \Delta_{gh} c\rangle\big)\propto \det \Delta_{gh},\qquad \Gamma\supset -\log\det \Delta_{gh}.$$

Итого (структура 1-loop эффективного действия):

$$\Gamma_{eff}=S_{cl}+\Gamma^{(1)}+O(\hbar^2),\qquad \Gamma^{(1)}=\tfrac12\log\det(\Delta_A)-\log\det(D\!\!\!\!/\,)-\log\det(\Delta_{gh}).$$

Замечание о нулевых модах: при наличии нулевых собственных значений корректно использовать $\det'$ (детерминант по ненулевым модам).

## 30.4 Оператор на 1-формах и разложение Ходжа

В ковариантной калибровке стандартная форма оператора на 1-формах:

$$\Delta_A^{MN}= -\nabla^2 g^{MN} + \Big(1-\frac{1}{\xi}\Big)\nabla^M\nabla^N + R^{MN}.$$

На компактном $K$ применяем разложение Ходжа для 1-форм:

$$A=A_{\perp}+d\phi,\qquad d^\*A_{\perp}=0.$$

На подпространстве продольных мод $d\phi$ соответствующая часть оператора сводится к скалярному лапласиану $-\nabla^2$.

На уровне спектра (с оговоркой о гармонических формах/нулевых модах) это выражается факторизацией:

$$\det'(\Delta_1)=\det'(\Delta_{1,\perp})\cdot\det'(\Delta_0).$$

Комбинация, возникающая в 1-loop части,

$$\tfrac12\log\det'(\Delta_1)-\log\det'(\Delta_0)$$

сводится к

$$\tfrac12\log\det'(\Delta_{1,\perp})-\tfrac12\log\det'(\Delta_0).$$

 То есть ghost-сектор реализует исключение калибровочных (продольных) степеней свободы и делает выражение калибровочно-инвариантным; оставшиеся тонкости (объём калибровочной группы, нулевые моды, выбор меры) кодируются в аккуратном обращении с $\det'$.

 Замечание (нулевые моды в $U(1)$ и почему это важно для $\kappa_{Cas}$): скалярный лапласиан $\Delta_0$ на компактном многообразии имеет нулевую моду (константные функции). В терминах калибровки это соответствует **остаточной глобальной калибровочной симметрии** $\omega=\mathrm{const}$ (для которой $A\mapsto A+d\omega$ не меняет поле). Стандартная процедура состоит из двух частей:

 1) в определении функционального детерминанта используется $\det'$ (нулевая мода исключается из спектрального произведения);

 2) одновременно деление на объём группы калибровки $\mathrm{Vol}(U(1))$ устраняет интегрирование по этой остаточной степени свободы.

 На произведении $K=\mathbb{RP}^3\times S^1$ это означает, что из совместного спектра $\lambda_{k,m}$ убирается только **одна** истинная 4D нулевая собственная величина $(k,m)=(0,0)$, тогда как KK-ряд $(k,m)=(0,m\neq 0)$ остаётся и даёт 1D Casimir-остаток из §30.6.

 Для этого же $K=\mathbb{RP}^3\times S^1$ полезно зафиксировать топологическое число нулевых мод: над $\mathbb{R}$ имеем $b_0=1$ (одна константная скалярная мода) и $b_1=1$ (одна гармоническая 1-форма, порождённая $d\theta$ вдоль $S^1$). Поэтому:

 1) в скалярном ghost-секторе всегда есть ровно одна нулевая мода (константный параметр глобальной калибровки);

 2) в 1-форменном секторе есть нулевая мода, соответствующая плоской связности вдоль $S^1$ (модуль плоских соединений); она не устраняется локальным калибровочным преобразованием и должна учитываться не через $\det$, а через конечномерный интеграл по модулю.

 Это и есть место, где целочисленный сдвиг $\zeta_{\det'}(0)=a_2-N_0$ (см. §30.8) должен компенсироваться нормировкой меры (объём остаточной группы и интеграл по гармоническим модам), иначе возникает искусственная $\log\mu$-неоднозначность.

 Лемма (гармоническая 1-мода и конечномерный фактор меры): на $K=\mathbb{RP}^3\times S^1$ разложение Ходжа в 1-формах имеет вид
 $$A=A_{\perp}+d\phi + A_{harm},\qquad A_{harm}=a\,d\theta,\qquad d^\*A_\perp=0,$$
 где $d\theta$ — (с точностью до нормировки) единственная гармоническая 1-форма, а коэффициент $a$ параметризует голономию вдоль $S^1$.
 В евклидовом действии Maxwell гармоническая компонента даёт $F(A_{harm})=0$, поэтому она не должна входить в бесконечномерный гауссов интеграл, определяющий $\det$; вместо этого она учитывается конечномерным интегралом по модулю плоских связностей (фундаментальная область фиксируется большими калибровками).
 Более явно: для $U(1)$ большие калибровки на $S^1$ имеют вид $\omega(\theta)=n\theta$ (в смысле $e^{i\omega}$ однозначно), поэтому $A\mapsto A+d\omega$ сдвигает $a\mapsto a+n$, $n\in\mathbb{Z}$. Следовательно, интеграл по модулю берётся по фундаментальной области $a\in[0,1)$ (эквивалентно — по одному классу голономии), и даёт конечный $\mu$-независимый множитель.
 Этот конечномерный множитель относится к **нормировке** $Z$ и является $\mu$-независимым; Casimir-остаток $\kappa_{Cas}$ возникает не из него, а из бесконечномерной части (KK-ряд по $m\neq 0$ для константной моды на $\mathbb{RP}^3$).

 Следствие (почему фактор $\mathrm{Vol}(S^1)$ не обязан оставаться в калибровочной нормировке): пусть $y$ — координата на $S^1$ длины $L=\mathrm{Vol}(S^1)$. Тогда класс больших калибровок имеет вид $e^{i\omega(y)}$ с $\omega(y)=2\pi n\,y/L$, и гармоническая мода может быть записана как
 $$A_{harm}=\frac{2\pi a}{L}\,dy,\qquad a\sim a+n,\qquad a\in[0,1).$$
 То есть физически интегрируется не «вся прямая» постоянных $A_y$, а **фундаментальная область**, длина которой по переменной $A_y$ равна $2\pi/L$.

 Этот факт фиксирует, что единственная безразмерная комбинация, в которой может появляться $L$ в нормировке калибровочного сектора, имеет вид $g_5^2/L$ (эквивалентно — $g_5^2/\mathrm{Vol}(S^1)$). Поэтому условие вида $g_5^2=\mathrm{Vol}(S^1)$ в `08_jacobian_derivation.md`, §8.1 можно трактовать не как произвольный постулат, а как **выбор стандартной нормировки $U(1)$-связности**, согласованный с большими калибровками/периодичностью Wilson-line.

 Лемма (почему это фиксирует нормировку, а не вводит “ручку”): в $U(1)$-теории калибровочное поле $A$ есть связность на главном $U(1)$-расслоении и нормируется так, что голономия вдоль любой замкнутой петли имеет вид
 $$\mathrm{Hol}_\gamma(A)=\exp\Big(i\int_\gamma A\Big),\qquad \int_\gamma A\in 2\pi\,\mathbb{Z}\ \text{для тривиальной связности и больших калибровок}.$$
 В частности, для генератора $S^1$ это означает периодичность переменной Wilson-line $\theta:=\int_{S^1}A$ по сдвигу $\theta\sim\theta+2\pi$.
 Если в теории присутствует материя с **минимальным зарядом** (в RPFT это соответствует “минимальной заряженной моде” $n=1$), то допустимые представления $U(1)$ фиксируют решётку зарядов $q\in\mathbb{Z}$, и связность $A$ нельзя произвольно рескейлить $A\mapsto \lambda A$ без одновременного изменения этой решётки (иначе меняется периодичность $\theta$ и спектр больших калибровок).
 Следовательно, при фиксированной нормировке группы $U(1)$ и фиксированной единице заряда единственная возможная размерная свобода в 5D Maxwell-секторе действительно сводится к одному числу $g_5^2$ (а на $S^1$ — к безразмерной комбинации $g_5^2/L$).
 В строгой ветке RPFT дополнительный безразмерный параметр в калибровочном секторе не вводится, поэтому принимается стандартная нормировка
 $$\frac{g_5^2}{\mathrm{Vol}(S^1)}=1\qquad\Longleftrightarrow\qquad g_5^2=\mathrm{Vol}(S^1).$$

## 30.5 Дзета-регуляризация детерминанта и heat kernel

Для положительного самосопряжённого эллиптического оператора $O$ с дискретным спектром $\{\lambda_n\}$ и кратностями $\{d_n\}$:

$$\zeta_O(s)=\sum_n \frac{d_n}{\lambda_n^s}.$$

Связь с heat kernel:

$$K_O(t)=\mathrm{Tr}(e^{-tO})=\sum_n d_n e^{-t\lambda_n},\qquad \zeta_O(s)=\frac{1}{\Gamma(s)}\int_0^\infty t^{s-1}K_O(t)\,dt.$$

Дзета-регуляризованный детерминант:

$$\log\det(O):=-\zeta'_O(0).$$

Для операторов лапласовского типа на $d$-мерном компактном многообразии (без края) при $t\to 0^+$:

$$K_O(t)\sim \frac{1}{(4\pi t)^{d/2}}\sum_{k=0}^{\infty} a_k(O)\,t^{k/2},\qquad a_0(O)=\int d\mathrm{vol}\,\mathrm{tr}(\mathbf{1}).$$

Эта асимптотика объясняет, почему ведущий вклад в спектрально-регуляризованный $\log\det$ является экстенсивным по объёму (через $a_0$), а подведущие поправки выражаются через локальные геометрические инварианты (коэффициенты $a_k$).

Замечание о масштабе: в общей постановке присутствует выбор масштаба регуляризации $\mu$ (схематично $\log\det_\mu O=-\zeta'_O(0)-\zeta_O(0)\log\mu^2$).

## 30.6 Как возникает параметр $\kappa_{Cas}$

 В RPFT Casimir-поправка записывается в виде
 $$\delta_{Cas}=\frac{\kappa_{Cas}}{S_{geo}}.$$
 Здесь $\kappa_{Cas}$ — **безразмерная константа**, описывающая константную (неэкстенсивную) часть 1-loop вклада после нормировки на геометрическое ядро $S_{geo}$.

 На интуитивном (и стандартном в КТП/finite-temperature) уровне связь с Casimir возникает так: если один из факторов геометрии является окружностью $S^1$ длины $L$, то эффективное действие содержит вклад вакуумной энергии
 $$\Gamma^{(1)}\supset L\,E_{vac}(S^1)+\cdots,$$
 где для **одной** безмассовой степени свободы на $S^1$ спектр частот
 $$\omega_n=\frac{2\pi}{L}|n|,\qquad n\in\mathbb{Z},$$
 и ζ-регуляризация даёт
 $$E_{vac}(S^1)=\frac12\sum_{n\in\mathbb{Z}}\omega_n=\frac{2\pi}{L}\sum_{n\ge 1}n=\frac{2\pi}{L}\,\zeta_R(-1)=-\frac{\pi}{6L}.$$
 При $L=2\pi$ получаем $E_{vac}(S^1)=-1/12$.

 Далее важна 1-loop **комбинация детерминантов** для калибровочного сектора (см. §30.4): ghost-сектор и Ходж-разложение приводят к тому, что продольные моды сокращаются, и остаётся эффективная степень детерминанта
 $$\tfrac12\log\det'(\Delta_{1,\perp})-\tfrac12\log\det'(\Delta_0).$$
 То есть в калибровочном секторе появляется «половинная» степень детерминанта, согласованная с общим фактором $1/2$ для бозонных гауссианов. В RPFT используется значение
 $$\kappa_{Cas}= -\frac{\zeta_R(-1)}{2}=\frac{1}{24}\qquad\text{(значение, используемое в численной проверке)},$$
 которое мотивируется связью с 1D Casimir-константой $\zeta_R(-1)=-1/12$ и указанной выше «половинной» структурой det.

### Лемма (строгая 1D-нормировка): $-\zeta_R(-1)/2$ как Abel/heat-kernel остаток

 Рассмотрим 1D сумму (Abel-суммирование)
 $$S(t):=\sum_{n\ge 1} n\,e^{-tn},\qquad t>0.$$
 Тогда (геометрическая прогрессия)
 $$S(t)=\frac{e^{-t}}{(1-e^{-t})^2}.$$ 
 Для $t\to 0^+$ имеет место стандартная асимптотика Бернулли (эквивалентная разложению $x/(e^x-1)$):
 $$S(t)=\frac{1}{t^2}-\frac{1}{12}+O(t^2).$$
 Следовательно, существует конечный 1D-остаток
 $$\kappa_{Cas}:=-\frac12\lim_{t\to 0^+}\Big(S(t)-\frac{1}{t^2}\Big)=\frac{1}{24}.$$
 При этом, так как Abel-сумма совпадает с ζ-регуляризацией для полиномиальных расходимостей, получаем эквивалентную запись
 $$\kappa_{Cas}=-\frac{\zeta_R(-1)}{2},\qquad \zeta_R(-1)=-\frac{1}{12}.$$ 
 Эта лемма фиксирует именно **1D-нормировку константного остатка** (после вычитания ведущей $1/t^2$-расходимости); в калибровочном секторе QED дополнительный множитель $1/2$ согласован с комбинацией
 $$\tfrac12\log\det'(\Delta_{1,\perp})-\tfrac12\log\det'(\Delta_0)$$
 из §30.4.

 Замечание (почему здесь именно $\zeta_R$, а не дзета Гурвица): в ghost/калибровочном секторе параметр калибровки $\omega$ является функцией на $K$ со значениями в $U(1)$ (локально — вещественный скаляр, но глобально — отображение $K\to U(1)$). Поэтому вдоль $S^1$ он обязан быть **периодическим** (однозначность калибровочного преобразования). Следовательно, одномерный KK-ряд для соответствующего лапласиана использует частоты $m\in\mathbb{Z}$ и даёт именно $\zeta_R(-1)$ (а не сдвинутый спектр и не $\zeta(\cdot,a)$).

 Замечание (как это входит на $K=\mathbb{RP}^3\times S^1$): для скалярного лапласиана на произведении спектр (формально) факторизуется
 $$\lambda_{k,m}=\lambda_k(\mathbb{RP}^3)+\Big(\tfrac{2\pi m}{L}\Big)^2,\qquad m\in\mathbb{Z},$$
 где $\lambda_0(\mathbb{RP}^3)=0$ соответствует константной гармонике на $\mathbb{RP}^3$.
 При переходе к $\det'$ убирается только **единственная** 4D нулевая собственная величина $(k,m)=(0,0)$, но при этом остаётся полный KK-ряд $(k,m)=(0,m\neq 0)$, который и даёт 1D Casimir-остаток.
 Численно это проявляется тем, что в калибровочной комбинации (после Hodge+ghost) вклад всех уровней $\lambda_k(\mathbb{RP}^3)>0$ почти полностью сокращается, а константный остаток совпадает с $-\zeta_R(-1)/2=1/24$ (см. прототип KK в `02_zeta_compute.py`, §4c).

 Замечание (почему именно уровень $\lambda_{\mathbb{RP}^3}=0$): в Maxwell+ghost комбинации
 $$\tfrac12\log\det'(\Delta_{1,\perp})-\tfrac12\log\det'(\Delta_0)$$
 на $\mathbb{RP}^3$ нет гармонических коэкзактных 1-форм (нулевой моды в $\Delta_{1,\perp}$), тогда как в скалярном секторе есть константная мода $\lambda_0(\mathbb{RP}^3)=0$. Поэтому в “массово-нульовом” секторе по $\mathbb{RP}^3$ остаётся именно 1D KK-ряд по $S^1$ (моды $m\in\mathbb{Z}\setminus\{0\}$) с общей степенью $-1/2$, что и приводит к константному остатку $-\zeta_R(-1)/2$.
 Для уровней $\lambda_k(\mathbb{RP}^3)>0$ соответствующие 1D вклады являются “массивными” по параметру $a=\sqrt{\lambda_k}$ и подавляются при $L=2\pi$ экспоненциально по $e^{-La}$ (это и наблюдается в KK-прототипе `02_zeta_compute.py`, §4c).

## 30.7 Что нужно доказать строго для $\kappa_{Cas}=1/24$

 В текущей версии строгой ветки следующие пункты являются критическими:

 1) корректная связь между «Casimir-подобной» константной частью 1-loop эффективного действия и выражением через ζ-регуляризацию детерминантов (включая выбор масштаба $\mu$ и вычитание экстенсивных членов);

 2) аккуратный учёт нулевых мод и фактора объёма калибровочной группы (переход от $\det$ к $\det'$);

 3) вывод того, что в комбинации QED детерминантов остаётся ровно «половинный» вклад, эквивалентный $-\zeta_R(-1)/2$ при $L=2\pi$;

 4) согласование этой константы с нормировкой, принятой в $S_{geo}$ (то есть почему именно деление на $S_{geo}$ соответствует выбранной безразмерной схеме).

## 30.8 Отрицательный (условный) результат: почему $\kappa_{Cas}$ может быть не фиксирован без доп. условия

Ключевая тонкость: в ζ-регуляризации детерминант обычно определяют с масштабом $\mu$:
$$\log\det\!\left(\frac{O}{\mu^2}\right):=-\zeta'_O(0)-\zeta_O(0)\,\log\mu^2.$$
Отсюда при замене $\mu\mapsto e^t\mu$ константная часть сдвигается на
$$\Delta\log\det = -2t\,\zeta_O(0).$$
В размерности $d=4$ величина $\zeta_O(0)$ выражается через heat-kernel коэффициент $a_2(O)$, то есть является **локальным** (геометрическим/операторным) вкладом. Следовательно:

1) без фиксации схемы/условия нормировки (выбор $\mu$ и вычитаний локальных контрчленов) константная часть 1-loop эффективного действия, а значит и параметр $\kappa_{Cas}$, вообще говоря **не является однозначным числом**;

 2) даже при фиксированном $\mu$ возможна зависимость от выбора спин-структуры/граничных условий вдоль $S^1$ (периодические vs антипериодические моды), что меняет “одномерную” сумму и может заменить $\zeta_R(-1)$ на значение через дзета-функцию Гурвица.

 В RPFT мы фиксируем физическую постановку как вакуумную QED на компактном внутреннем пространстве $K=\mathbb{RP}^3\times S^1$, где окружность $S^1$ является **пространственной KK-окружностью**, а не евклидовым временем термальной статистической суммы. Поэтому для фермионов на $S^1$ выбираем периодическую spin-структуру/BC:
 $$\Psi(\theta+2\pi)=\Psi(\theta).$$
 Антипериодический выбор $\Psi(\theta+2\pi)=-\Psi(\theta)$ соответствует термальному (KMS) следу и является другой физической задачей. Численно, однако, сравнение P/AP полезно как тест устойчивости константной части 1-loop (см. оценку $\pm 0.155\sigma$ ниже).

 Замечание (численная чувствительность Dirac к BC на $S^1$): в прототипе KK на $K=\mathbb{RP}^3\times S^1$ (см. `02_zeta_compute.py`, §4c) калибровочная комбинация Maxwell+ghost даёт $\kappa_{Cas}^{gauge}=1/24$ с точностью $\sim 10^{-8}$ и полностью доминируется уровнем $\lambda_{\mathbb{RP}^3}=0$. При этом для 4D безмассового Dirac остаётся малый не-локальный остаток по KK-частоте на $S^1$ (в ζ-det-подобной форме):
 $$F_{Dirac}(P)\approx +1.808\cdot 10^{-6},\qquad F_{Dirac}(AP)\approx -1.808\cdot 10^{-6}.$$
 Если формально трактовать его как сдвиг $\kappa\mapsto \kappa+F_{Dirac}$, то это даёт
 $$\Delta\alpha^{-1}\approx -\frac{F_{Dirac}}{S_{geo}}\approx \pm 1.32\cdot 10^{-8}\ \ (\approx \pm 0.155\sigma\ \text{от CODATA}).$$
 Это подчёркивает, что хотя $\kappa_{Cas}^{gauge}$ стабилен и фиксируется калибровочным сектором, **полная** константная часть 1-loop эффективного действия может слабо зависеть от выбора BC/спин-структуры для фермионов на $S^1$, если этот выбор не закреплён отдельным принципом.

 Таким образом, утверждение вида «$\kappa_{Cas}=1/24$ следует из первых принципов без дополнительных соглашений» может оказаться ложным: в лучшем случае получается **условная теорема** вида

 > Если фиксированы (i) схема ζ-детерминанта (масштаб $\mu$ и правило вычитания локальных членов), (ii) спин-структура на $S^1$, (iii) обращение с нулевыми модами ($\det'$), и (iv) именно QED-комбинация операторов после gauge fixing/ghost/Hodge, то $\kappa_{Cas}$ вычисляется как определённый спектральный инвариант. В RPFT используется кандидат $\kappa_{Cas}=-\zeta_R(-1)/2=1/24$.

### Сильный вариант (проверка): на $\mathbb{RP}^3\times S^1$ исчезает стандартная 4D геометрия, отвечающая за $\log\mu$

Для стандартной метрики произведения на $\mathbb{RP}^3\times S^1$ (радиусы $R=1$) обнуляются два ключевых 4D инварианта:

- $W^2$ (норма тензора Вейля),
- $E_4$ (плотность Эйлера; интеграл даёт $32\pi^2\chi$).

Точный расчёт $W^2=0$ и $E_4=0$ для этого фона приведён в `10_a2_coefficient.md`, §10.5.1.

Это важно потому, что для **конформных** 4D полей (Dirac/Maxwell) коэффициент перед $\log\mu$ в 1-loop эффективном действии выражается через интегралы комбинаций $W^2$ и $E_4$ (с точностью до полной производной). Если на фоне эти инварианты обнуляются, то остаётся шанс, что в полной QED-комбинации детерминантов $\log\mu$-неоднозначность реально сокращается, и тогда параметр $\kappa_{Cas}$ становится схемонезависимым.

### Следствие (если принять стандартные формулы для конформных аномалий): вклад $\log\mu$ исчезает

По определению ζ-детерминанта с масштабом $\mu$ (см. начало §30.8):
$$\log\det\!\left(\frac{O}{\mu^2}\right)=-\zeta'_O(0)-\zeta_O(0)\,\log\mu^2,$$
то есть коэффициент при $\log\mu$ равен $-2\,\zeta_O(0)$.

Для операторов лапласовского типа на 4D многообразии величина $\zeta_O(0)$ выражается через соответствующий heat-kernel коэффициент порядка $t^0$ (в нашей нотации — через $a_2(O)$).

Для **конформных** 4D полей (массовый параметр равен нулю) стандартный результат (в форме локальной конформной аномалии) имеет вид
$$\zeta_O(0)=\frac{1}{(4\pi)^2}\int_{\mathbb{RP}^3\times S^1}\!\big( c\,W^2-a\,E_4\big)\,d\mathrm{vol}$$
с точностью до полной производной $\nabla^2 R$ (которая даёт ноль после интегрирования на замкнутом многообразии при гладкой метрике и постоянной $R$).

 В частности, для безмассовых Dirac и Maxwell (после корректного gauge fixing + ghost, т.е. для калибровочно-инвариантной комбинации детерминантов) коэффициент при $\log\mu$ пропорционален комбинации интегралов $\int W^2$ и $\int E_4$.

 На нашем фоне $\mathbb{RP}^3\times S^1$ с метрикой произведения имеем $W^2=0$ и $E_4=0$ (см. `10_a2_coefficient.md`, §10.5.1), следовательно
 $$\zeta_O(0)=0\quad\Rightarrow\quad \text{коэффициент при }\log\mu\text{ равен }0.$$

  Более конкретно: в стандартной нормировке конформной аномалии в 4D (см., например, обзоры Duff; Birrell–Davies) для безмассового 4D спинора Дирака и для Maxwell (вместе с ghost-сектором) выполняется
 $$\zeta_O(0)=\frac{1}{(4\pi)^2}\int (c\,W^2-a\,E_4)\,d\mathrm{vol},$$
 где коэффициенты $(a,c)$ — фиксированные числа (Dirac: $a=\tfrac{11}{360}$, $c=\tfrac{1}{10}$; Maxwell+ghost: $a=\tfrac{31}{180}$, $c=\tfrac{1}{10}$). Поэтому на нашем фоне, где $W^2\equiv 0$ и $E_4\equiv 0$ **точечно**, получаем $\zeta_O(0)=0$ для каждого из этих конформных секторов и тем самым отсутствие $\log\mu$ без подгонки.

 Тонкость (нулевые моды и $\det'$): если у $O$ есть нулевые собственные значения и используется $\det'$, то в ζ-подходе появляется сдвиг
 $$\zeta_{O,\,\det'}(0)=a_2(O)-N_0,$$
 где $N_0$ — число нулевых мод. Поэтому даже при $a_2(O)=0$ (например, из-за $W^2=E_4=0$) может остаться **целочисленный** вклад в коэффициент при $\log\mu$.
 В калибровочно-инвариантной постановке этот вклад должен быть компенсирован стандартным обращением с объёмом остаточной калибровочной группы и/или интегрированием по гармоническим модам (см. замечание о $\mathrm{Vol}(U(1))$ и $\det'$ в §30.4). Именно это место нужно оформить строго, иначе «$\kappa_{Cas}$ фиксирован схемонезависимо» не следует.

 Лемма (нулевые моды, $\det'$ и $\log\mu$): в функциональном интеграле нулевые моды не интегрируются «гауссово» и не должны входить в бесконечномерный $\det$. Если определять $\det'\!\left(O/\mu^2\right)$ ζ-методом, то исключение $N_0$ нулевых мод даёт целочисленный сдвиг $\zeta(0)$ и, соответственно, множитель $\mu^{+2N_0}$ (в логарифме — добавку $+N_0\log\mu^2$).
 Однако в калибровочном секторе QED после Hodge+ghost возникает именно комбинация
 $$\tfrac12\log\det'(\Delta_{1,\perp})-\tfrac12\log\det'(\Delta_0),$$
 и целочисленные $\det'$-сдвиги в коэффициенте при $\log\mu$ в ней сокращаются пропорционально разности $b_0-b_1$ (числа нулевых мод скаляра и 1-форм). Для $K=\mathbb{RP}^3\times S^1$ имеем $b_0=b_1=1$, поэтому этот возможный «целочисленный» источник $\log\mu$ исчезает уже на уровне самой детерминантной комбинации.
 При этом конечномерные факторы (деление на $\mathrm{Vol}(U(1))$ для глобальной калибровки и интеграл по гармонической компоненте $A_{harm}\propto d\theta$ как модулю плоских связностей) всё равно необходимы для корректной нормировки $Z$, но они относятся к конечномерной части меры и не вводят дополнительной схемной $\mu$-неоднозначности.

 Итого для QED на $K=\mathbb{RP}^3\times S^1$ с метрикой произведения (радиусы $R=1$) при калибровочно-инвариантной комбинации операторов (Maxwell+ghost в форме §30.4 и безмассовый Dirac) коэффициент при $\log\mu$ равен нулю: локальный вклад исчезает из-за $W^2\equiv 0$ и $E_4\equiv 0$, а целочисленные $\det'$-сдвиги в калибровочном секторе сокращаются, поскольку $b_0=b_1=1$; для Dirac нулевых мод на этом фоне нет.

 В эквивалентной форме: коэффициент при $\log\mu^2$ равен $-\zeta_{tot,\,\det'}(0)$, где $\zeta_{O,\,\det'}(0)=\zeta_O(0)-N_0$.
 Для калибровочного сектора важна именно калибровочно-инвариантная комбинация Maxwell+ghost (см. §30.4), и для неё (как для конформного 4D поля) имеем
 $$\zeta_{Maxwell+gh}(0)=\frac{1}{(4\pi)^2}\int (c\,W^2-a\,E_4)\,d\mathrm{vol}=0$$
 на $K=\mathbb{RP}^3\times S^1$, поскольку $W^2\equiv 0$ и $E_4\equiv 0$.
 Далее, при переходе к $\det'$ появляется дополнительный **целочисленный** сдвиг из-за исключения нулевых мод. Для комбинации
 $$\tfrac12\log\det'(\Delta_{1,\perp})-\tfrac12\log\det'(\Delta_0)$$
 этот сдвиг равен $\tfrac12(b_0-b_1)$; на нашем фоне $b_0=b_1=1$, значит нулемодовый вклад в $\zeta_{gauge,\,\det'}(0)$ также равен нулю.
 Для безмассового Dirac на $\mathbb{RP}^3\times S^1$ при $R=6>0$ нулевых мод нет по формуле Лихнеровича $D^2=\nabla^\*\nabla+R/4$, а конформный (локальный) вклад при $W^2=E_4=0$ даёт $\zeta_{Dirac}(0)=0$. Следовательно $\zeta_{tot,\,\det'}(0)=0$.

 Это не вычисляет саму константную часть (т.е. не даёт автоматически $\kappa_{Cas}=1/24$), но убирает главный источник схемозависимости: остаётся вычислить уже **схемоинвариантную** конечную часть комбинации детерминантов на $\mathbb{RP}^3\times S^1$ при фиксированной спин-структуре вдоль $S^1$.
