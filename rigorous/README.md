## Ключевой результат

**Формула:**
```
α⁻¹ = S_geo - δ_Cas - δ_BB
    = (4π³ + π² + π) - 1/(24·S_geo) - C/(π⁴·S_geo²)
```

где:
- **S_geo** — **ВЫВОД** из 1-loop функционального интеграла (Теоремы 5.2–5.4)
- **Почему сумма** — log det(O₁·O₂) = log det O₁ + log det O₂
- **δ_Cas = 1/24** — вывод из heat kernel (Gilkey)
- **δ_BB: форма 1/π⁴** — вывод (π⁴ = Vol(RP³)²)
- **δ_BB: коэффициент C = 1** — наблюдение (отклонение 0.64%)

## Открытые вопросы

1. **Коэффициент C = 1** — требует 2-loop вычисления на L(2,1)×S¹ (отклонение 0.64%)
2. **Выбор K = RP³ × S¹** — постулат (минимальность обоснована в §1.1)
