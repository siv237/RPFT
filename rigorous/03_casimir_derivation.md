# §3. ВЫВОД ЧЛЕНА 1/24 ИЗ СПЕКТРАЛЬНОЙ ГЕОМЕТРИИ

**Цель:** Показать, что $\delta_{Casimir} = 1/(24 \cdot S_{geo})$ следует из спектральной суммы, а не из "интерпретации решётки Лича".

---

## 3.1 Casimir энергия: общая теория

### Определение

Энергия вакуумных флуктуаций квантового поля на компактном многообразии $M$:
$$E_{Cas} = \frac{1}{2} \sum_n \omega_n = \frac{1}{2} \sum_n \sqrt{\lambda_n}$$

где $\lambda_n$ — собственные значения лапласиана $\Delta$.

### Регуляризация (дзета-функция)

Сумма расходится. Регуляризуем через спектральную дзета-функцию:
$$\zeta_\Delta(s) = \sum_n \lambda_n^{-s}$$

Тогда:
$$E_{Cas} = \frac{1}{2} \zeta_\Delta(-1/2)$$

или через производную:
$$\log \det(\Delta) = -\zeta'_\Delta(0)$$

---

## 3.2 Эталонный пример: окружность $S^1$

### Спектр

На $S^1$ длины $L = 2\pi$:
$$\lambda_n = n^2, \quad n = 0, \pm 1, \pm 2, \ldots$$

Кратность: $d_n = 2$ для $n \neq 0$, $d_0 = 1$.

### Дзета-функция

$$\zeta_{S^1}(s) = 2 \sum_{n=1}^\infty n^{-2s} = 2 \zeta_R(2s)$$

где $\zeta_R$ — дзета Римана.

### Casimir энергия

При нормировке длины $L=2\pi$:
$$E_{Cas}(S^1) = \frac{1}{2} \zeta_{S^1}(-1/2) = \zeta_R(-1) = -\frac{1}{12}.$$

**Важно:** Коэффициент $-1/12$ — это $\zeta_R(-1)$.

### Связь с 1/24

Для **безмассового** скаляра с периодическими граничными условиями:
Пусть окружность имеет длину $L$; тогда собственные частоты
$$\omega_n = \frac{2\pi}{L}|n|,\qquad n\in\mathbb{Z}.$$
Регуляризованная вакуумная энергия:
$$E_{Cas}(S^1)=\frac12\sum_{n\in\mathbb{Z}}\omega_n
=\frac{2\pi}{L}\sum_{n=1}^{\infty} n
=\frac{2\pi}{L}\,\zeta_R(-1)
=-\frac{\pi}{6L}.$$
В частности, при $L=2\pi$ получаем $E_{Cas}=-1/12$.

**Замечание (почему иногда появляется $1/24$):** если по тем или иным причинам в спектре остаётся *половина мод* (например, «хиральная» половина спектра), то
$$E_{Cas}^{\mathrm{half}}(S^1)=\frac12\sum_{n=1}^{\infty}\frac{2\pi}{L}n=\frac{\pi}{L}\zeta_R(-1)=-\frac{\pi}{12L},$$
и при $L=2\pi$ это даёт $-1/24$.

**Честный статус:** для связи с коэффициентом $1/24$ в формуле для $\alpha^{-1}$ требуется строго показать, почему в рассматриваемом детерминанте возникает именно «половинный» (или эквивалентный ему по нормировке) вклад.

## 3.3 Обобщение на 3-сферу $S^3$

### Спектр скаляров на $S^3$

$$\lambda_n = n(n+2), \quad d_n = (n+1)^2$$

### Дзета-функция

$$\zeta_{S^3}(s) = \sum_{n=1}^\infty \frac{(n+1)^2}{[n(n+2)]^s}$$

### Асимптотика при больших n

При $n \to \infty$:
- $\lambda_n \sim n^2$
- $d_n \sim n^2$

Поэтому:
$$\zeta_{S^3}(s) \sim \sum_n n^{2-2s} \sim \zeta_R(2s-2)$$

### Вычисление $\zeta'_{S^3}(0)$

Используем формулу Эйлера-Маклорена или разложение heat kernel.

**Теорема (Dowker 1977):**
$$\zeta'_{S^3}(0) = -\frac{1}{90} + \text{(конечные члены)}$$

## 3.4 Casimir на L(2,1) = RP³

### Спектр (только чётные n)

$$\zeta_{L(2,1)}(s) = \sum_{k=1}^\infty \frac{(2k+1)^2}{[2k(2k+2)]^s}$$

### Связь с $\zeta_{S^3}$

$$\zeta_{L(2,1)}(s) = \frac{1}{2} \left[ \zeta_{S^3}(s) + \eta_{S^3}(s) \right]$$

где $\eta$ — "чётная часть" спектра.

### Численное вычисление

```python
from mpmath import mp, nsum, diff, pi, inf
mp.dps = 50

def zeta_L21(s):
    def term(k):
        n = 2*k
        d = (n+1)**2
        lam = n*(n+2)
        return d / lam**s
    return nsum(term, [1, inf])

# При s → 0: ζ(s) расходится, но ζ'(0) конечна (регуляризация)
# Используем heat kernel метод
```

## 3.5 Heat Kernel и коэффициент 1/24

### Heat trace

$$K(t) = \text{Tr}(e^{-t\Delta}) = \sum_n d_n e^{-t\lambda_n}$$

### Асимптотика при $t \to 0$

$$K(t) \sim \frac{1}{(4\pi t)^{d/2}} \sum_{k=0}^\infty a_k t^k$$

где $a_k$ — коэффициенты Сили-ДеВитта (Seeley-DeWitt).

Для 3-мерного многообразия:
- $a_0 = \text{Vol}(M)$
- $a_1 = \frac{1}{6} \int_M R \, d\text{vol}$ (интеграл скалярной кривизны)
- $a_2 = $ (более сложные инварианты)

### Связь с $\zeta'(0)$

$$\zeta'(0) = -\gamma a_{d/2} + \int_0^\infty \frac{dt}{t} \left[ K(t) - \sum_{k < d/2} a_k t^{k-d/2} \right]$$

где $\gamma$ — постоянная Эйлера.

### Для L(2,1) × S¹ (4D)

Здесь важно не смешивать две разные величины:

- **Casimir энергия** (вакуумная энергия) выражается через $\zeta_\Delta(-1/2)$.
- **Функциональный детерминант** в 1-loop эффективном действии выражается через $\zeta'_\Delta(0)$:
$$\log\det(\Delta)= -\zeta'_{\Delta}(0).$$

Поправка вида «$1/24$» в RPFT относится к *константному* (scale-dependent) вкладу, возникающему при ζ-регуляризации детерминанта на произведении $L(2,1)\times S^1$ и последующей нормировке на $S_{geo}$.

**Честный статус:** прямое равенство вида $\zeta'_{M}(0)=-1/24$ для $M=L(2,1)\times S^1$ не доказано в этом файле; для полного строгого вывода требуется согласовать нормировки (включая выбор масштаба $\mu$) и вычислить соответствующую комбинацию heat-kernel коэффициентов для операторов QED.

## 3.6 Утверждение (структура и что осталось доказать)

### Утверждение 3.1 (Casimir-поправка в формуле RPFT)

В формуле RPFT используется поправка
$$\delta_{Cas}=\frac{\kappa_{Cas}}{S_{geo}},\qquad \kappa_{Cas}=\frac{1}{24}\ \text{(используется в численной проверке)}.$$
Её интерпретация: «Casimir-подобный» вклад (константная часть после ζ-регуляризации и фиксации нормировки), делённый на геометрическое ядро $S_{geo}$.

*Доказательство:*

**Шаг 1.** Heat kernel на произведении факторизуется:
$$K_M(t) = K_{L(2,1)}(t) \cdot K_{S^1}(t)$$

**Шаг 2.** Heat kernel на $S^1$ длины $L = 2\pi$ (Jacobi theta function):
$$K_{S^1}(t) = \frac{L}{\sqrt{4\pi t}} \sum_{n=-\infty}^{\infty} e^{-n^2 L^2 / (4t)} = \frac{L}{\sqrt{4\pi t}} \theta_3(0, e^{-L^2/(4t)})$$

При $t \to 0$ доминирует $n = 0$:
$$K_{S^1}(t) \approx \frac{L}{\sqrt{4\pi t}} = \frac{2\pi}{\sqrt{4\pi t}}$$

**Шаг 3.** Heat kernel на $L(2,1)$ (Weyl + поправки):
$$K_{L(2,1)}(t) = \frac{\text{Vol}(L(2,1))}{(4\pi t)^{3/2}} \left[1 + \frac{R}{6} t + O(t^2)\right]$$

где $R = 6$ — скалярная кривизна единичной сферы.

**Шаг 4.** Произведение:
$$K_M(t) = \frac{\text{Vol}(L(2,1)) \cdot L}{(4\pi t)^2} \left[1 + t + O(t^2)\right]$$

 **Шаг 5.** Связь с $\zeta'(0)$ через формулу Меллина:
 $$\zeta(s) = \frac{1}{\Gamma(s)} \int_0^\infty t^{s-1} K(t) \, dt$$
 
 При $s\to 0$ аналитическое продолжение определяет конечную часть $-\zeta'(0)$ (то есть $\log\det$). В размерности $d=4$ эта конечная часть зависит от коэффициента $a_2$ и выбора масштаба $\mu$ в нормировке $\log\det(P/\mu^2)$ (см. `10_a2_coefficient.md`, §10.7).

**Шаг 6.** Коэффициент $a_2$ для произведения $M^3 \times S^1$:

По теореме Гилки (Gilkey 1975, Theorem 4.1.6):
$$a_2(M \times S^1) = a_2(M) \cdot L + a_1(M) \cdot a_1(S^1) + a_0(M) \cdot a_2(S^1)$$

Для $S^1$: $a_1(S^1) = 0$ (плоская), $a_2(S^1) = 0$.
Для $L(2,1)$: $a_2(L(2,1))$ вычисляется через кривизну.

**Шаг 7.** Финальный результат:
 
 На этом шаге требуется **согласовать нормировки** между 1D Casimir-энергией на $S^1$ (опирающейся на $\zeta_R(-1)$) и тем, какая именно комбинация детерминантов (операторов QED) входит в 1-loop эффективное действие на $L(2,1)\times S^1$.
 
 Поэтому в рамках данного файла мы фиксируем лишь структуру: из ζ/heat-kernel следует, что константный вклад в детерминант даёт поправку вида $\delta_{Cas}\propto 1/S_{geo}$, а точный коэффициент (в частности $1/24$) требует отдельного нормировочного расчёта.
 
 ∎

**Честный статус:** вывод корректен по структуре, но строгая нормировка коэффициента $1/24$ требует отдельного расчёта (см. также `10_a2_coefficient.md`, `04_heat_kernel.py`, `13_casimir_explicit.py`).

## 3.8 Почему именно 24?

### Математическое объяснение

24 появляется из:
1. $\zeta_R(-1) = -1/12$ (дзета Римана)
2. $\chi(T^2) = 0$, но $\chi(S^2) = 2$
3. Центральный заряд бозонной струны: $c = 24$

 ### Геометрическое объяснение
 
 Для 4-мерного многообразия коэффициент Сили-ДеВитта $a_2$ (безразмерный при $R=1$):
 $$a_2 = \frac{1}{360} \int_M (c_1 R^2 + c_2 R_{\mu\nu}^2 + c_3 R_{\mu\nu\rho\sigma}^2) \, \frac{d\text{vol}}{R^4}$$
 
 где $c_1, c_2, c_3$ зависят от типа поля.
 
 Для $S^3 \times S^1$ при $R=1$ (планковские единицы) $a_2$ масштабируется как безразмерная комбинация интегралов кривизны и объёма. Однако выделение конкретного численного коэффициента (например $1/24$) требует:
 
 - выбора конкретного оператора (скаляр/вектор/спинор и калибровочная фиксация),
 - согласования нормировки ζ-регуляризованного детерминанта (масштаб $\mu$),
 - и учёта комбинации детерминантов, которая реально входит в 1-loop QED.
 
 Строгий вывод коэффициента $1/24$ требует полного расчёта комбинации детерминантов и согласования нормировок; heat kernel контролирует структуру через коэффициенты $a_k$.

### Связь с решёткой Лича

Решётка Лича $\Lambda_{24}$ — уникальная 24-мерная чётная унимодулярная решётка.

В теории струн: $D_{crit} - 2 = 26 - 2 = 24$ поперечных измерений.
 
 **Но:** Это **интерпретация**, а не вывод. Строгий вывод требует полного расчёта комбинации детерминантов и согласования нормировок.
