# СТРОГИЙ ВЫВОД ЧЛЕНА π

**Проблема:** Член π в формуле α⁻¹ = 4π³ + π² + π — наименее обоснованный.

**Решение:** π = расстояние в M_flat(RP³, U(1)) — топологический инвариант!

---

## ГЛАВНЫЙ РЕЗУЛЬТАТ

### Пространство модулей плоских связностей

Для U(1) bundle над RP³:

$$\mathcal{M}_{flat}(\text{RP}^3, U(1)) = \text{Hom}(\pi_1(\text{RP}^3), U(1)) / U(1)$$

Поскольку π₁(RP³) = Z₂:

$$\text{Hom}(Z_2, U(1)) = \{e^{i\theta} : e^{2i\theta} = 1\} = \{1, -1\} = \{0, \pi\}$$

**M_flat = {0, π} — ровно две точки!**

### Расстояние в M_flat

$$d(0, \pi) = |\pi - 0| = \pi$$

**Это ТОПОЛОГИЧЕСКИЙ ИНВАРИАНТ, не феноменология!**

---

## ТРИ НЕЗАВИСИМЫХ АРГУМЕНТА

| Аргумент | Формула | Результат |
|----------|---------|-----------|
| **Систола** | L_sys(RP³) = π | π |
| **Голономия** | θ_max − θ_min = π − 0 | π |
| **M_flat** | d(θ=0, θ=π) = π | π |

**Все три дают π независимо!** Это следствие Z₂ структуры RP³.

---

## ПОЧЕМУ ИМЕННО π (НЕ 2π, π/2)?

### 1. Систола

RP³ = S³/Z₂, где Z₂ — антиподное отображение.
Расстояние до антипода = **половина большого круга** = π.

### 2. Z₂ голономия

U(1) bundle с Z₂ ограничением: exp(2iθ) = 1
Решения: θ ∈ {0, π}

### 3. M_flat

Для L(p,1): spacing = 2π/p

| p | L(p,1) | Spacing | = π? |
|---|--------|---------|------|
| 2 | RP³ | 2π/2 = **π** | ✅ |
| 3 | L(3,1) | 2π/3 ≈ 2.09 | ❌ |
| 4 | L(4,1) | 2π/4 ≈ 1.57 | ❌ |

**Член π СПЕЦИФИЧЕН для RP³!**

---

## ИЕРАРХИЯ РАЗМЕРНОСТЕЙ

```
4π³ = Vol(S³×S¹)  — 4D объём (L⁴)
π² = Vol(RP³)     — 3D объём (L³)
π = L_sys(RP³)    — 1D длина (L¹)
```

Каждый член — **интеграл соответствующей размерности**.
В планковских единицах (L = 1) все становятся числами.

## ФИЗИЧЕСКАЯ ИНТЕРПРЕТАЦИЯ

### Wilson loop

$$W_\gamma = \exp\left(i \oint_\gamma A\right)$$

Для двух вакуумов:
- W_γ(θ=0) = 1 (тривиальный bundle)
- W_γ(θ=π) = −1 (нетривиальный bundle)

**Фермионы требуют нетривиальный bundle** (антипериодичность)!

### Туннельный вклад (WKB)

$$S_{tunnel} = L_{sys} \times p_{min} = \pi \times 1 = \pi$$

### Топологическая информация

π = π × (1 бит) — каждый нестягиваемый цикл добавляет "1 бит" информации.
