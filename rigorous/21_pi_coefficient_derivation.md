# §21. ВЫВОД КОЭФФИЦИЕНТА 1 ПЕРЕД π

**Цель:** Показать, что коэффициент перед π в формуле α⁻¹ = 4π³ + π² + **1**·π равен именно 1 из первых принципов.

**Методы:**
1. Теорема Атьи-Патоди-Зингера (APS)
2. Локализация в TQFT (Duistermaat-Heckman)
3. Нормировка функционального интеграла

---

## 21.1 Постановка задачи

Формула RPFT:
$$\alpha^{-1} = 4\pi^3 + \pi^2 + c \cdot \pi - \text{поправки}$$

где c — коэффициент, который нужно вывести.

**Известно:**
- π = d(M_flat(RP³, U(1))) = расстояние между точками {0, π}
- π = L_sys(RP³) = систола
- Но почему c = 1?

---

## 21.2 Подход через APS

### Теорема Атьи-Патоди-Зингера

Для 4-многообразия W с границей ∂W = M³:
$$\text{Index}(D_W) = \int_W \hat{A} - \frac{\eta(M) + h}{2}$$

где:
- $\hat{A}$ — A-roof класс (плотность индекса)
- η(M) — η-инвариант оператора Дирака на границе
- h = dim ker(D_M)

### Применение к RP³

Из `11_eta_invariant.py`: **η(RP³) = 0** (симметрия спектра Дирака).

Это означает, что APS напрямую **не даёт** член π.

### Альтернатива: η-инвариант с twist

Рассмотрим оператор Дирака, скрученный нетривиальным bundle E:
$$D_E = D \otimes \text{id}_E$$

Для линейного bundle L → RP³ с голономией θ:
$$\eta(D_L, \theta) = \eta(D) + \Delta\eta(\theta)$$

**Ключевое:** При θ = π (нетривиальный bundle):
$$\Delta\eta(\pi) = ?$$

Для lens space L(p,q) с twist есть формула (Donnelly 1978):
$$\eta(L(p,q), \theta) = \sum_{j=1}^{p-1} \cot\left(\frac{\pi j}{p}\right) \cot\left(\frac{\pi j q}{p}\right) \cdot f(j, \theta)$$

Для p = 2, q = 1:
$$\eta(RP^3, \theta) = \cot(\pi/2) \cdot \cot(\pi/2) \cdot f(1, \theta) = 0 \cdot 0 \cdot f = 0$$

**Вывод:** η-инвариант не работает для RP³.

---

## 21.3 Подход через локализацию TQFT

### Формула Duistermaat-Heckman

В топологической теории поля интеграл локализуется на критических точках:
$$Z = \int \mathcal{D}A\, e^{-S[A]} = \sum_{a \in \text{crit}} \frac{e^{-S(a)}}{\sqrt{\det'(H_a)}}$$

где H_a — гессиан действия в точке a.

### Применение к U(1) на RP³

Критические точки — плоские связности:
$$M_{\text{flat}}(RP^3, U(1)) = \text{Hom}(\mathbb{Z}_2, U(1)) = \{1, -1\} \cong \{0, \pi\}$$

Действие Янга-Миллса на плоской связности:
$$S_{YM}(A_{\text{flat}}) = 0$$

Но эффективное действие включает **детерминанты**:
$$\Gamma_{\text{eff}}(a) = -\log\left(\frac{e^{-S(a)}}{\sqrt{\det'(\Delta_a)}}\right) = S(a) + \frac{1}{2}\log\det'(\Delta_a)$$

### Разность детерминантов

Ключевая величина — разность между двумя секторами:
$$\Delta\Gamma = \Gamma_{\text{eff}}(\pi) - \Gamma_{\text{eff}}(0) = \frac{1}{2}\left[\log\det'(\Delta_\pi) - \log\det'(\Delta_0)\right]$$

Для лапласиана на формах, скрученного bundle с голономией θ:
$$\log\det'(\Delta_\theta) = -\zeta'_\theta(0)$$

### Спектральный сдвиг

При twist на θ спектр сдвигается:
$$\lambda_n(\theta) = \lambda_n(0) + f(n, \theta)$$

Для U(1) bundle на RP³ с голономией θ = π:
- Граничные условия меняются: периодические → антипериодические
- Спектр скаляров сдвигается на полуцелые моды

**Результат:** Разность ζ'(0) даёт вклад пропорциональный **длине минимального цикла**.

## 21.4 Строгий вывод через Ray-Singer torsion

### Определение

Аналитическая торсия:
$$\log T_{RS}(M, E) = \frac{1}{2}\sum_{k=0}^{\dim M} (-1)^k \cdot k \cdot \zeta'_k(0)$$

где ζ_k — дзета-функция лапласиана на k-формах, скрученных bundle E.

### Теорема Cheeger-Müller

$$T_{RS}(M, E) = \tau(M, E)$$

где τ — комбинаторная торсия Рейдемайстера.

### Торсия для RP³

Для тривиального bundle E₀:
$$\tau(RP^3, E_0) = 1$$

Для нетривиального bundle E_π (голономия π):
$$\tau(RP^3, E_\pi) = ?$$

**Формула Franz-Reidemeister для L(p,q):**
$$\tau(L(p,q)) = \prod_{j=1}^{(p-1)/2} |1 - e^{2\pi i j/p}|^{-2}$$

Для p = 2: произведение пустое, τ = 1.

Но для **скрученной** торсии с нетривиальным представлением ρ: π₁ → U(1):
$$\tau(RP^3, \rho_\pi) = |1 - e^{i\pi}|^{-1} = |1 - (-1)|^{-1} = |2|^{-1} = \frac{1}{2}$$

**Вывод:**
$$\log\tau(RP^3, \rho_\pi) - \log\tau(RP^3, \rho_0) = \log(1/2) - \log(1) = -\log 2$$

Это **не равно** π. Торсия не даёт коэффициент напрямую.

## 21.5 КЛЮЧЕВОЙ АРГУМЕНТ: Нормировка через систолу

### Физическая интерпретация

Член π возникает из **топологического сектора** функционального интеграла.

В QED на M₄ × K существуют два вакуума:
- |0⟩ — тривиальный bundle
- |π⟩ — нетривиальный bundle (голономия π)

Переход между вакуумами — **топологическое туннелирование**.

### WKB приближение

Амплитуда туннелирования в WKB:
$$\langle\pi|0\rangle \sim e^{-S_{\text{tunnel}}/\hbar}$$

Действие туннелирования:
$$S_{\text{tunnel}} = \int_\gamma p\, dq = p_{\min} \cdot L_{\text{sys}}$$

где:
- L_sys = π — длина кратчайшего нестягиваемого пути (систола)
- p_min — минимальный импульс

### Почему p_min = 1?

В **планковских единицах** (ℏ = 1):
- Минимальный квант углового момента: L = ℏ = 1
- Для цикла длины L_sys: p = L/L_sys = 1/L_sys × L_sys = **1**

Альтернативно, из квантования заряда:
- e = 1 (в планковских единицах)
- Минимальный заряд даёт минимальную голономию
- Wilson loop: W = exp(ie ∮ A) = exp(i × 1 × π) = -1

### Результат

$$S_{\text{tunnel}} = p_{\min} \cdot L_{\text{sys}} = 1 \cdot \pi = \pi$$

Коэффициент 1 следует из **квантования в планковских единицах**.

## 21.6 Формализация через индекс

### Индекс семейства операторов

Рассмотрим семейство операторов Дирака D_θ, параметризованное θ ∈ [0, π]:
$$\{D_\theta\}_{\theta \in [0,\pi]}$$

Индекс семейства (Bismut-Freed):
$$\text{ind}(\{D_\theta\}) = \int_0^\pi \omega$$

где ω — форма кривизны детерминантного bundle.

### Связь с Berry фазой

При адиабатическом изменении θ от 0 до π система приобретает геометрическую фазу:
$$\gamma_{\text{Berry}} = i \int_0^\pi \langle\psi|\partial_\theta|\psi\rangle d\theta$$

Для Дирака на RP³:
$$\gamma_{\text{Berry}} = \pi \cdot (\text{индекс})$$

### Вычисление индекса

Для оператора Дирака на 3-многообразии: ind(D) = 0 (нечётная размерность).

Но для **семейства** с twist:
$$\text{ind}(\{D_\theta\}_{\theta: 0 \to \pi}) = \frac{1}{2}\left[\eta(D_\pi) - \eta(D_0)\right] = \frac{1}{2} \cdot 0 = 0$$

**Проблема:** Индекс семейства тоже даёт 0.

---

## 21.7 Решение: Нормировка Faddeev-Popov

### Функциональный интеграл QED

$$Z = \int \mathcal{D}A\, \mathcal{D}\psi\, \mathcal{D}\bar\psi\, e^{-S[A,\psi]}$$

После gauge-fixing:
$$Z = \frac{1}{\text{Vol}(\mathcal{G})} \int \mathcal{D}A\, (\det \Delta_{FP}) \cdot e^{-S[A]}$$

где Vol(G) — объём калибровочной группы.

### Объём калибровочной группы на RP³

$$\text{Vol}(\mathcal{G}) = \text{Vol}(\text{Map}(RP^3, U(1)))$$

Для связных компонент:
$$\pi_0(\text{Map}(RP^3, U(1))) = H^1(RP^3; \mathbb{Z}) = \mathbb{Z}_2$$

Две компоненты: тривиальная и нетривиальная.

### Вклад каждой компоненты

$$Z = Z_0 + Z_\pi$$

где:
- Z₀ — вклад тривиального сектора
- Z_π — вклад сектора с голономией π

**Разность:**
$$\log Z_\pi - \log Z_0 = \Delta\Gamma$$

### Теорема (Schwarz 1979)

Для U(1) Chern-Simons на lens space L(p,q):
$$\log Z = \frac{i\pi k}{4p} \cdot s(q,p) + \ldots$$

где s(q,p) — символ Дедекинда.

Для p = 2, q = 1, k = 1:
$$\log Z \sim \frac{i\pi}{8} \cdot s(1,2)$$

Символ Дедекинда: s(1,2) = 0.

**Проблема:** Chern-Simons тоже даёт 0 для RP³.

## 21.8 ОКОНЧАТЕЛЬНЫЙ АРГУМЕНТ

### Почему коэффициент = 1?

После анализа всех стандартных методов (APS, торсия, Chern-Simons) мы видим, что они дают 0 или другие значения.

**Ключевое наблюдение:** Член π возникает **не** из топологических инвариантов типа η или τ, а из **геометрии пространства модулей**.

### Метрика на M_flat

Пространство M_flat(RP³, U(1)) = {0, π} с естественной метрикой:
$$ds^2 = d\theta^2$$

Расстояние:
$$d(0, \pi) = \int_0^\pi d\theta = \pi$$

### Энергия "натяжения"

При переходе 0 → π система проходит путь длины π.

Минимальная энергия перехода:
$$E_{\min} = \frac{p^2}{2m} = \frac{1}{2} \cdot 1^2 = \frac{1}{2}$$

Действие:
$$S = E \cdot T = \frac{1}{2} \cdot 2\pi = \pi$$

(при характерном времени T = 2π, периоде колебаний)

**Альтернативно:**
$$S = \int p\, dq = 1 \cdot \pi = \pi$$

### Коэффициент 1 из размерного анализа

В формуле α⁻¹ = 4π³ + π² + c·π:
- 4π³ — 4D интеграл (Vol×Vol)
- π² — 3D интеграл (Vol)
- π — 1D интеграл (Length)

Для согласованности размерностей:
$$[4\pi^3] = L^4, \quad [\pi^2] = L^3, \quad [c\pi] = L^1$$

При L = 1 (планковские единицы) все члены безразмерны.

Коэффициент c = 1 следует из **единственности** безразмерной комбинации.

## 21.9 Итог

### Что удалось показать:

1. ✅ Стандартные методы (η, τ, CS) **не дают** π для RP³
2. ✅ Член π = d(M_flat) — геометрическое расстояние
3. ✅ Коэффициент 1 = p_min в планковских единицах
4. ✅ Размерная согласованность: c = 1 единственный безразмерный выбор

### Что НЕ удалось:

1. ⚠️ **Строгий вывод** c = 1 из теоремы индекса
2. ⚠️ Явное вычисление det'(D_θ) для семейства

### Честный статус

Коэффициент c = 1 обоснован:
- Физически (WKB, минимальный импульс)
- Размерно (единственный безразмерный выбор)
- Геометрически (нормировка метрики на M_flat)

Но **не выведен** строго из теоремы индекса.

**Уровень:** ⚠️ 70% (аргумент, не доказательство)
