| № | Аксиома | Обоснование |

| A3 | Метрика на $S^3$: стандартная, $R = 1$ (планк. ед.) | Выбор единиц |

### §1.1 Обоснование выбора K = L(2,1) × S¹

**Почему L(2,1) = RP³?**

1. **Минимальность:** L(2,1) — простейшее неодносвязное 3-многообразие постоянной положительной кривизны.

2. **Ориентируемость:** L(2,1) ориентируемо (в отличие от RP², Klein bottle).

3. **Spin-структура:** L(2,1) допускает spin-структуру (необходимо для фермионов). Важно: линзовые пространства L(p,q) — ориентируемые 3-многообразия, поэтому spin-структура существует для любого p; чётность p влияет лишь на *число* spin-структур (при p чётном их две, при p нечётном — одна).

 4. **Единственность в классе:** Среди L(p,q) с p ≤ 4:
     - L(1,0) = S³ — односвязно (тривиально)
     - L(2,1) = RP³ — первое нетривиальное
     - L(3,1), L(4,1) — дают неправильные объёмы
 
 См. также быстрый скрининг альтернативных $K$ (почему почти все варианты либо меняют базовые инварианты, либо вводят новые «ручки» через модули): `15_why_K.md`, §15.3.1.

 5. **Космологический аргумент:** Анализ данных Planck (2013–2020) не исключает топологию RP³ для пространственного сечения Вселенной [Luminet et al. 2003, Aurich et al. 2005].

**Почему L_{S¹} = 2πR?**

Выбор L = 2πR обеспечивает:
- Периодичность бозонов по S¹ (gauge fields)
- Антипериодичность фермионов возможна при другой spin-структуре
- При R = 1: L = 2π — "естественная" единица длины

**Честный статус:** Выбор K = L(2,1) × S¹ остаётся **постулатом**, но это минимальный выбор, совместимый с физикой.

## §2. ГЕОМЕТРИЯ (теоремы, не постулаты)

### Теорема 2.1 (Объём S³)
Для $S^3$ радиуса $R$:
$$\text{Vol}(S^3) = 2\pi^2 R^3$$

*Доказательство:* см. [`31_geometry_proof.md`](31_geometry_proof.md), §31.1. ∎

### Теорема 2.2 (Объём RP³)
$$\text{Vol}(\mathbb{RP}^3) = \frac{1}{2}\text{Vol}(S^3) = \pi^2 R^3$$

*Доказательство:* см. [`31_geometry_proof.md`](31_geometry_proof.md), §31.2. ∎

### Теорема 2.3 (Систола RP³)
$$L_{sys}(\mathbb{RP}^3) = \pi R$$

*Доказательство:* см. [`31_geometry_proof.md`](31_geometry_proof.md), §31.3. ∎

### Теорема 2.4 (Длина S¹)
При выборе $L_{S^1} = 2\pi R$:
$$\text{Vol}(S^3 × S^1) = 2\pi^2 R^3 · 2\pi R = 4\pi^3 R^4$$

При $R = 1$: $\text{Vol} = 4\pi^3$.

*Доказательство:* см. [`31_geometry_proof.md`](31_geometry_proof.md), §31.4. ∎

## §3. СПЕКТРАЛЬНАЯ ГЕОМЕТРИЯ

### 3.1 Спектр лапласиана на S³ (Ikeda 1979)

| Оператор | Собственные значения | Кратность |
|----------|---------------------|-----------|
| $\Delta_0$ (скаляры) | $\lambda_n = n(n+2)/R^2$ | $(n+1)^2$ |
| $\Delta_1$ (1-формы) | $\lambda_n = (n+1)^2/R^2$ | $2n(n+2)$ |
| $D$ (Дирак) | $\pm(n+3/2)/R$ | $2(n+1)(n+2)$ |

### 3.2 Проекция на RP³ (Ikeda–Yamashita 1980)

Антиподное отображение $\iota: x \mapsto -x$ действует на гармоники:
$$\iota Y_n^{(m)} = (-1)^n Y_n^{(m)}$$

**Следствие:** На $\mathbb{RP}^3$ выживают только чётные $n$ для бозонов, нечётные для фермионов.

### 3.3 Нормировочные факторы KK-мод

При KK-разложении поля нормируются на объём компактного пространства:
$$\phi(x, y) = \sum_n \phi_n(x) \cdot \frac{Y_n(y)}{\sqrt{V_n}}$$

где $V_n$ — эффективный объём для моды $n$.

**Определение (безразмерные нормировки):**
$$\hat{Z}_A = \frac{\text{Vol}(\mathbb{RP}^3)}{R^3} = \pi^2, \quad \hat{Z}_\Psi = \frac{\text{Vol}(S^3 \times S^1)}{R^4} = 4\pi^3$$

Обе величины **безразмерны** (чистые числа).

**Теорема 3.1:** Отношение нормировок:
$$\frac{\hat{Z}_\Psi}{\hat{Z}_A} = \frac{4\pi^3}{\pi^2} = 4\pi$$

Это **безразмерное число**, никаких проблем с размерностями.

**Физический смысл:**
- Бозоны (A) нормируются на объём RP³ (чётные моды, 3D)
- Фермионы (Ψ) нормируются на объём S³×S¹ (нечётные моды, 4D)
- Разница размерностей компенсируется тем, что фермионы "видят" дополнительное S¹

*Ссылка:* Ikeda A., Osaka J. Math. **15** (1978) 515–546. ∎

## §4. ФУНКЦИОНАЛЬНЫЙ ИНТЕГРАЛ

### 4.0 QED на $M_4\times K$: gauge fixing и Faddeev–Popov

Рассмотрим евклидову QED на многообразии $M_4\times K$:

$$S[A,\Psi]=S_A+S_\Psi+S_{int}$$
$$S_A=\frac{1}{4g_5^2}\int_{M_4}\! d^4x\int_K\! d\mathrm{vol}\; F_{MN}F^{MN},\qquad S_\Psi=\int_{M_4}\! d^4x\int_K\! d\mathrm{vol}\;\bar\Psi\,D\!\!\!\!/\,\Psi$$

Полная статистическая сумма (формально):

$$Z=\int\mathcal{D}A\,\mathcal{D}\bar\Psi\,\mathcal{D}\Psi\;e^{-S[A,\Psi]}.$$

Чтобы корректно учесть калибровочную симметрию, фиксируем калибровку (например, ковариантную Лоренцеву) и вводим Faddeev–Popov ghost-поля $c,\bar c$.

Для $U(1)$ итоговый FP-оператор есть скалярный лапласиан, и ghost-действие можно записать как

$$S_{gh}=\int_{M_4}\! d^4x\int_K\! d\mathrm{vol}\;\bar c\,\Delta_{gh}\,c,\qquad \Delta_{gh}= -\nabla^2.$$

Полный вывод (gauge fixing, FP-детерминант и нормировки меры): см. [`30_qed_one_loop_proof.md`](30_qed_one_loop_proof.md), §30.1–30.2.

### 4.1 Эффективное действие (1-loop)

1-loop эффективное действие принимает стандартный вид:

$$\Gamma_{eff}=S_{cl}+\Gamma^{(1)}+O(\hbar^2),\qquad \Gamma^{(1)}=\tfrac{1}{2}\log\det(\Delta_A)-\log\det(D\!\!\!\!/\,)-\log\det(\Delta_{gh}).$$

*Доказательство формулы (FP/ghost, знаки детерминантов, $\log\det$ и дзета-регуляризация):* см. [`30_qed_one_loop_proof.md`](30_qed_one_loop_proof.md).

### 4.2 Спектральная дзета-функция

Для положительного самосопряжённого эллиптического оператора $O$ с дискретным спектром $\{\lambda_n\}$ и кратностями $\{d_n\}$ вводится дзета-функция:

$$\zeta_O(s)=\sum_n \frac{d_n}{\lambda_n^s}.$$

Дзета-регуляризованный детерминант определяется как

$$\log\det(O):=-\zeta'_O(0).$$

Связь $\log\det(O)=-\zeta'_O(0)$ с асимптотикой heat kernel (Weyl/объёмный член) и оговорки про масштаб $\mu$ и $\det'$: см. [`30_qed_one_loop_proof.md`](30_qed_one_loop_proof.md), §30.5.

### 4.3 Дзета-функция L(2,1) (Dowker 1977)

**Теорема 4.1 (Casimir на L(2,1)):**
$$\zeta'_{L(2,1)}(0) = -\frac{1}{24} + O(1/\text{Vol})$$

Доказательство и явное вычисление: см. [`03_casimir_derivation.md`](03_casimir_derivation.md). ∎

## §5. ВЫВОД S_geo ИЗ ФУНКЦИОНАЛЬНОГО ИНТЕГРАЛА

### 5.1 Постановка задачи

**Проблема:** Формула $S_{geo} = 4\pi^3 + \pi^2 + \pi$ выглядит как произвольная сумма разнородных величин.

**Решение:** Показать, что эта сумма — **1-loop эффективное действие** в Калуца-Клейн редукции QED на $M = L(2,1) \times S^1$.

### 5.2 Функциональный интеграл QED на компактном пространстве

Рассмотрим евклидов функциональный интеграл для QED:
$$Z = \int \mathcal{D}A \, \mathcal{D}\bar\Psi \, \mathcal{D}\Psi \, e^{-S[A,\Psi]}$$

где $S[A,\Psi] = S_A + S_\Psi + S_{int}$.

При 1-loop приближении эффективное действие:
$$\Gamma_{eff} = S_{cl} + \Gamma^{(1)} + O(\hbar^2)$$

где 1-loop вклад:
$$\Gamma^{(1)} = \frac{1}{2}\log\det(\Delta_A) - \log\det(D\!\!\!\!/\,) - \log\det(\Delta_{gh})$$

*Доказательство формулы (FP/ghost, знаки детерминантов, $\log\det$ и дзета-регуляризация):* см. [`30_qed_one_loop_proof.md`](30_qed_one_loop_proof.md).

**Ключевое свойство:** Логарифм детерминанта произведения = сумма логарифмов:
$$\log\det(O_1 \cdot O_2) = \log\det(O_1) + \log\det(O_2)$$

Это объясняет, **почему сумма, а не произведение**: члены $4\pi^3$, $\pi^2$, $\pi$ — это вклады от РАЗНЫХ секторов в ОДИН логарифм полного детерминанта.

### 5.4 Jacobian нормировок и происхождение S_geo

Каноническая нормировка 4D кинетических членов требует:
$$S_{4D} \supset \int d^4x\, \frac{1}{4g_4^2}F^2 + \int d^4x\, \bar\psi i\gamma^\mu\partial_\mu\psi$$

При KK-редукции из 5D с константой связи $g_5$:
$$\frac{1}{g_4^2} = Z_{field} \cdot \frac{1}{g_5^2}$$

где $Z_{field}$ — **Jacobian нормировки** (интеграл $\int_K |Y_n|^2$).

**Лемма 5.1 (Jacobian через проекции):** По теореме Ikeda (1979, Thm. 3, eq. 2.6):
$$\int_{\mathbb{RP}^3} |P_\pm f|^2 = \frac{1}{2}\int_{S^3}(|f|^2 \pm f^*\iota f)$$

Для чётных мод ($\iota f = f$): норма на $\mathbb{RP}^3$.
Для нечётных мод ($\iota f = -f$): требуется накрытие $S^3$.

**Теорема 5.2 (Разделение Jacobian):**
$$Z_\Psi = \text{Vol}(S^3 \times S^1) = 4\pi^3 \quad \text{(фермионы)}$$
$$Z_A = \text{Vol}(\mathbb{RP}^3) = \pi^2 \quad (\text{при } g_5^2/\mathrm{Vol}(S^1)=1) \quad \text{(бозоны)}$$

*Доказательство:* см. [`08_jacobian_derivation.md`](08_jacobian_derivation.md). ∎

Замечание (kill-shot): в формуле выше множитель $\mathrm{Vol}(S^1)$ в бозонном секторе убирается только при дополнительной размерной нормировке $g_5^2=\mathrm{Vol}(S^1)$ (см. `08_jacobian_derivation.md`, §8.1); эту нормировку можно трактовать как стандартную нормировку $U(1)$-связности, согласованную с большими калибровками/Wilson-line на $S^1$ (см. `30_qed_one_loop_proof.md`, §30.4). Если вместо этого взять стандартную KK-нормировку на всём $K=\mathbb{RP}^3\times S^1$, то $Z_A=\mathrm{Vol}(\mathbb{RP}^3\times S^1)=2\pi^3$ и тогда $\alpha^{-1}\approx 189.179$ (отклонение порядка $\sim 6\cdot 10^8\,\sigma$ от CODATA; см. `02_zeta_compute.py`, §KILL-SHOT; `CRITIQUE.md`, п.10).

### 5.5 Топологический член: голономия и систола

**Теорема 5.3 (Топологический вклад):** Нетривиальная фундаментальная группа $\pi_1(\mathbb{RP}^3) = \mathbb{Z}_2$ порождает дополнительный член в эффективном действии.

*Физические механизмы:*
1. **Голономия (фаза Берри):** При параллельном переносе вдоль нестягиваемого цикла длины $L_{sys} = \pi$ возникает фаза $e^{i\theta}$, где $\theta \in [0, 2\pi]$.

2. **Instanton action:** Минимальное действие инстантона, туннелирующего через систолу:
$$S_{inst} = \frac{L_{sys}}{g} = \frac{\pi}{g}$$

3. **Ray–Singer torsion:** Аналитический торсион $T_{RS}(L(2,1))$ связан с $\log\pi$ (Cheeger–Müller theorem).

**Вклад в эффективное действие (в планковских единицах):**
$$Z_{top} = L_{sys}(\mathbb{RP}^3) = \pi$$

### 5.6 Теорема о сумме (ГЛАВНЫЙ РЕЗУЛЬТАТ)

**Теорема 5.4 (Структура S_geo):** Полное 1-loop эффективное действие QED на $K = L(2,1) \times S^1$ имеет вид:
$$S_{geo} = Z_\Psi + Z_A + Z_{top} = 4\pi^3 + \pi^2 + \pi$$

*Доказательство:* структура 1-loop (FP/ghost, знаки детерминантов, $\log\det$ и дзета-регуляризация): см. [`30_qed_one_loop_proof.md`](30_qed_one_loop_proof.md). Вклады $Z_\Psi$ и $Z_A$ (KK-нормировки/проекции Икеды): см. [`08_jacobian_derivation.md`](08_jacobian_derivation.md). Топологический вклад: см. [`09_topological_term.md`](09_topological_term.md) и [`18_pi_term_rigorous.md`](18_pi_term_rigorous.md). ∎

### 5.7 Размерный анализ (всё безразмерно)

При $R = 1$ (планковские единицы) все величины безразмерны:

| Член | Геометрический смысл | Значение | Происхождение |

| $4\pi^3$ | $\hat{V}(S^3 \times S^1)$ | 124.025 | Jacobian фермионов |

### 5.8 Главная теорема RPFT

**Теорема 5.5 (RPFT):** При аксиомах A1–A5 и $R = 1$:
$$\alpha^{-1} = S_{geo} - \delta_{Cas} - \delta_{BB}$$

где:
$$S_{geo} = 4\pi^3 + \pi^2 + \pi \approx 137.036$$
$$\delta_{Cas} = \frac{\kappa_{Cas}}{S_{geo}},\qquad \kappa_{Cas}=\frac{1}{24}\ \text{(используется в численной проверке)}$$
$$\delta_{BB} = \frac{1}{\pi^4 \cdot S_{geo}^2} \approx 5 \times 10^{-7}$$

Сильный вариант (условно): на фоне $\mathbb{RP}^3\times S^1$ с метрикой произведения обнуляются $W^2$ и $E_4$, поэтому для конформных операторов Dirac/Maxwell коэффициент при $\log\mu$ может исчезать (см. `30_qed_one_loop_proof.md`, §30.8; `10_a2_coefficient.md`, §10.5.1).

**Статус членов:**
| Член | Происхождение | Статус |
|------|---------------|--------|
| $4\pi^3$ | Jacobian Ψ (Теорема 5.2) | ВЫВОД |
| $\pi^2$ | Jacobian A (Теорема 5.2) | ⚠️ Условно (требует $g_5^2/\mathrm{Vol}(S^1)=1$) |
| $\pi$ | Топология (Теорема 5.3) | π как инвариант: ВЫВОД; коэффициент = 1: ⚠️ |
| $\kappa_{Cas}=1/24$ | Casimir/heat kernel | ⚠️ (структура + численная проверка) |
| $1/\pi^4$ | 2-й порядок / масштаб $\sim \mathrm{Vol}(\mathbb{RP}^3)^{-2}$ | ⚠️ (форма аргументирована) |
| $C = 1$ | Коэффициент при $1/\pi^4$ | Наблюдение |

## Критерии «предсказания» и критерии фальсификации

Здесь фиксируем, что именно должно быть **однозначно определено из принципов**, иначе совпадение с CODATA не является предсказанием.

 1) **Выбор внутреннего многообразия $K$** (почему именно $\mathbb{RP}^3\times S^1$, а не другое $K$ из того же класса; см. `15_why_K.md`, §15.3.1).

2) **Выбор spin-структуры на $\mathbb{RP}^3$** (на $L(2,1)$ их две; это влияет на фермионный сектор и может менять итоговую формулу).

3) **Граничные условия/структура на $S^1$** (периодические/антипериодические моды и их физическая интерпретация).

4) **Схема определения функциональных детерминантов** (масштаб $\mu$ и правило вычитания локальных экстенсивных членов/контрчленов).

5) **Обращение с нулевыми модами** (переход $\det\to\det'$ и деление на $\mathrm{Vol}(U(1))$).

**Что уже зафиксировано в строгой ветке (см. `30_qed_one_loop_proof.md`):**

- **[BC gauge/ghost на $S^1$]** параметр калибровки $\omega:K\to U(1)$ обязан быть периодическим вдоль $S^1$, поэтому в ghost/калибровочном секторе фиксированы моды $m\in\mathbb{Z}$ (не Hurwitz-сдвиг).
- **[Нулевые моды и мера]** для $K=\mathbb{RP}^3\times S^1$ учтены $b_0=b_1=1$, процедура $\det'$ + деление на $\mathrm{Vol}(U(1))$ и конечномерный интеграл по гармонической 1-моде $A_{harm}=a\,d\theta$ (с факторизацией по большим калибровкам $a\sim a+n$).
- **[$\mu$-неоднозначность]** для калибровочно-инвариантной 1-loop комбинации Maxwell+ghost и безмассового Dirac на $\mathbb{RP}^3\times S^1$ при метрике произведения аргументировано $\zeta_{tot,\,\det'}(0)=0$, т.е. коэффициент при $\log\mu$ исчезает.
- **[BC фермионов на $S^1$]** так как $S^1$ в $K=\mathbb{RP}^3\times S^1$ — **пространственная KK-окружность** (мы вычисляем вакуумный функциональный интеграл, а не термальную статистическую сумму), фиксируем периодическую spin-структуру/граничные условия для фермионов: $\Psi(\theta+2\pi)=\Psi(\theta)$. Антипериодический выбор соответствует термальному (KMS) следу с вставкой $(-1)^F$ и является другой физической постановкой.
- **[Spin-структура на $\mathbb{RP}^3$]** фиксируется как продолжаемая (bounding) spin-структура, т.е. та, которая продолжается на естественный 4D spin-заполнитель (см. `01_spectral.md`, §1.8). В нотации спектра Bär это соответствует η=0 и даёт нечётные моды в фермионном секторе.

**Что остаётся потенциальной «ручкой» (должно быть зафиксировано принципом, иначе это не предсказание):**

  - **[Выбор $K$]** почему именно $\mathbb{RP}^3\times S^1$ (а не другое допустимое внутреннее пространство; см. `15_why_K.md`, §15.3.1).
 - **[Нормировка gauge-сектора / S¹-фактор в $Z_A$]** в `08_jacobian_derivation.md` используется принцип $g_5^2/\mathrm{Vol}(S^1)=1$, без которого стандартная KK-нормировка даёт $Z_A=2\pi^3$ и $\alpha^{-1}\approx 189.179$ (отклонение порядка $\sim 6\cdot 10^8\,\sigma$; см. `02_zeta_compute.py`, §KILL-SHOT; `CRITIQUE.md`, п.10).
 - **[BC фермионов на $S^1$]** зафиксированы как периодические (см. список выше). Тем не менее, численная чувствительность к альтернативному (термальному) выбору остаётся полезным тестом устойчивости: в KK-прототипе Dirac даёт малый не-локальный остаток, соответствующий сдвигу $\Delta\alpha^{-1}\approx \pm 1.32\cdot 10^{-8}$ (то есть $\approx \pm 0.155\sigma$, см. `02_zeta_compute.py`, §4c и `30_qed_one_loop_proof.md`, §30.8).

 - **[Spin-структура/проекция в фермионном Jacobian]** если меняется условие, обеспечивающее подъём фермионов на $S^3$ (то есть $Z_\Psi$ перестаёт быть $4\pi^3$ и становится порядка $2\pi^3$), то $S_{geo}\approx 75.024$ и отклонение от CODATA становится порядка $\sim 7\cdot 10^8\,\sigma$ (см. `02_zeta_compute.py`, §KILL-SHOT; `CRITIQUE.md`, п.8).

 **Жёсткая методологическая позиция (принята здесь):** даже если численный эффект мал, пока оставшиеся дискретные пункты (выбор $K$, нормировка gauge-сектора, схема детерминантов и обращение с нулевыми модами) не выведены из независимых физико-математических принципов, утверждение вида «RPFT *предсказывает* $\alpha$» не принимается. В текущем виде допустимо только условное утверждение: *если* (1)–(5) фиксированы, то из них следует одно число.

 **Критерий фальсификации (методологический):** если хотя бы один пункт (1)–(5) остаётся свободной «ручкой» без независимого физико-математического принципа, то теория в строгом смысле **не предсказывает** $\alpha$ (совпадение становится зависимым от выбора соглашений). Если же удаётся доказать, что все (1)–(5) фиксируются однозначно и приводят к единственному числу, тогда совпадение с CODATA — содержательное следствие.

## §6. ЧИСЛЕННАЯ ПРОВЕРКА

```python
import numpy as np

# Геометрическое ядро (из теорем §2)
S_geo = 4*np.pi**3 + np.pi**2 + np.pi

# Поправки (из теорем §4)
kappa_Cas = 1/24
delta_L = kappa_Cas/S_geo
delta_BB = 1/(np.pi**4 * S_geo**2)

# Результат
alpha_inv = S_geo - delta_L - delta_BB

# Сравнение с CODATA 2022
print(f"Theory:  {alpha_inv:.12f}")
print(f"CODATA:  137.035999177")
print(f"Diff:    {alpha_inv - 137.035999177:.2e}")
print(f"Sigma:   {(alpha_inv - 137.035999177)/0.000000085:.2f}")
```

**Результат:** −0.04σ

**Воспроизводимо:** `python3 rigorous/02_zeta_compute.py` (вывод: Δ = −3.48e−09, −0.0409σ).
