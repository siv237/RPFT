# §10. ВЫЧИСЛЕНИЕ КОЭФФИЦИЕНТА a₂ И ПРОИСХОЖДЕНИЕ 1/24

**Цель:** Явно вычислить коэффициент Seeley-DeWitt a₂ для M = L(2,1) × S¹ и показать связь с 1/24.

---

## 10.1 Формула Gilkey для a₂

### Общая формула (Gilkey 1975, Theorem 4.1.6)

Для оператора Лапласа на d-мерном римановом многообразии без границы:
$$a_2 = \frac{1}{(4\pi)^{d/2}} \int_M \left( \frac{R^2 - 3R_{ij}R^{ij} + R_{ijkl}R^{ijkl}}{180} \right) d\text{vol}$$

Для скалярного поля коэффициенты:
- c₁ = 1 (R²)
- c₂ = -3 (Ricci²)
- c₃ = 1 (Riemann²)

### Для спинорного поля

$$a_2^{spinor} = \frac{1}{(4\pi)^{d/2}} \int_M \left( \frac{-R^2/30 + R_{ij}R^{ij}/12 + R_{ijkl}R^{ijkl}/180} \right) d\text{vol}$$

---

## 10.2 Кривизна S³

### Метрика единичной S³

$$ds^2 = d\chi^2 + \sin^2\chi (d\theta^2 + \sin^2\theta \, d\phi^2)$$

### Тензор кривизны

Для сферы постоянной кривизны K = 1/R²:
$$R_{ijkl} = K (g_{ik}g_{jl} - g_{il}g_{jk})$$

При R = 1:
$$R_{ijkl} = g_{ik}g_{jl} - g_{il}g_{jk}$$

### Скалярные инварианты для S³

- **Скалярная кривизна:** R = d(d-1)K = 3·2·1 = 6
- **Ricci тензор:** R_{ij} = (d-1)K g_{ij} = 2g_{ij}
- **Ricci²:** R_{ij}R^{ij} = (d-1)²K²d = 4·1·3 = 12
- **Riemann²:** R_{ijkl}R^{ijkl} = 2d(d-1)K² = 2·3·2·1 = 12

---

## 10.3 Кривизна RP³ = L(2,1)

RP³ — факторпространство S³ по Z₂, локально изометрично S³.

**Следствие:** Все локальные инварианты кривизны те же:
- R = 6
- R_{ij}R^{ij} = 12
- R_{ijkl}R^{ijkl} = 12

**Объём:** Vol(RP³) = Vol(S³)/2 = π²

---

## 10.4 Кривизна S¹

S¹ — плоское 1-многообразие:
- R = 0
- R_{ij} = 0
- R_{ijkl} = 0

**Объём:** Vol(S¹) = 2π (при R = 1)

---

## 10.5 Кривизна произведения L(2,1) × S¹

Для произведения M × N:
- R(M×N) = R(M) + R(N) = 6 + 0 = 6
- Ricci: блочно-диагональный
- Riemann: блочно-диагональный

### Интегралы по M = L(2,1) × S¹

$$\int_M R \, d\text{vol} = R(RP^3) \cdot \text{Vol}(RP^3 \times S^1) = 6 \cdot \pi^2 \cdot 2\pi = 12\pi^3$$

$$\int_M R_{ij}R^{ij} \, d\text{vol} = 12 \cdot 2\pi^3 = 24\pi^3$$

$$\int_M R_{ijkl}R^{ijkl} \, d\text{vol} = 12 \cdot 2\pi^3 = 24\pi^3$$

---

## 10.6 Вычисление a₂ для M = L(2,1) × S¹

### Для скаляра (d = 4)

$$a_2^{scalar} = \frac{1}{(4\pi)^2} \cdot \frac{1}{180} \left[ R^2 - 3 R_{ij}R^{ij} + R_{ijkl}R^{ijkl} \right] \cdot \text{Vol}$$

Подставляем:
$$a_2^{scalar} = \frac{1}{16\pi^2} \cdot \frac{1}{180} \left[ 36 - 3 \cdot 12 + 12 \right] \cdot 2\pi^3$$

$$= \frac{1}{16\pi^2} \cdot \frac{1}{180} \cdot (36 - 36 + 12) \cdot 2\pi^3$$

$$= \frac{1}{16\pi^2} \cdot \frac{12}{180} \cdot 2\pi^3$$

$$= \frac{1}{16\pi^2} \cdot \frac{1}{15} \cdot 2\pi^3$$

$$= \frac{2\pi^3}{240\pi^2} = \frac{\pi}{120}$$

### Для спинора (d = 4)

Другие коэффициенты, но структура похожа.

---

## 10.7 Связь с ζ'(0) и Casimir

### Формула

$$\zeta'(0) = -\frac{1}{2} a_2 \cdot (\text{лог. множитель})$$

Для 4D:
$$-\zeta'(0) = a_2 + O(\text{конечное})$$

### Откуда 1/24?

Связь с ζ_R(-1) = -1/12:

Casimir energy на S¹:
$$E_{Cas}(S^1) = \frac{1}{2} \zeta(-1/2) = \zeta_R(-1) = -\frac{1}{12}$$

Для 4D многообразия нормировка:
$$\delta_{Cas} = -\frac{1}{2} \cdot \left(-\frac{1}{12}\right) = \frac{1}{24}$$

### Альтернативный вывод

Для произведения M³ × S¹:
$$a_2(M \times S^1) = a_2(M) \cdot L_{S^1} + a_{3/2}(M) \cdot a_{1/2}(S^1) + a_1(M) \cdot a_1(S^1) + ...$$

При L_{S¹} = 2π и a_k(S¹) = 0 для k > 0:
$$a_2(M \times S^1) = 2\pi \cdot a_2(M)$$

Для RP³:
$$a_2(RP^3) = \frac{1}{(4\pi)^{3/2}} \cdot \frac{R_{3D}}{6} \cdot \text{Vol}(RP^3) = \frac{1}{8\pi^{3/2}} \cdot 1 \cdot \pi^2 = \frac{\pi^{1/2}}{8}$$

**Примечание:** Размерный анализ сложен, нужна аккуратность с нормировками.

## 10.8 Численная проверка

```python
from mpmath import mp, pi
mp.dps = 50

# Инварианты кривизны для RP³
R = 6
Rij2 = 12
Rijkl2 = 12
Vol_RP3_S1 = pi**2 * 2*pi  # = 2π³

# Коэффициент a₂ для скаляра на 4D
a2_integrand = (R**2 - 3*Rij2 + Rijkl2) / 180
# = (36 - 36 + 12) / 180 = 12/180 = 1/15

a2 = a2_integrand * Vol_RP3_S1 / (4*pi)**2
# = (1/15) * 2π³ / 16π²
# = 2π / 240 = π/120

print(f"a₂ = {float(a2):.10f}")
print(f"a₂ / Vol = {float(a2 / Vol_RP3_S1):.10f}")
```

## 10.9 Статус вывода 1/24

| Шаг | Статус | Комментарий |
|-----|--------|-------------|
| Формула Gilkey | ✅ | Стандартная теория |
| Кривизна RP³ | ✅ | K = 1, R = 6 |
| Вычисление a₂ | ✅ | Прямой расчёт |
| **Связь a₂ ↔ 1/24** | ⚠️ | Нужна аккуратная нормировка |

### Честный ответ

Коэффициент 1/24 появляется из:
1. ζ_R(-1) = -1/12 (Casimir на S¹)
2. Множитель 1/2 от регуляризации
3. → 1/(2 × 12) = 1/24

Формула Gilkey даёт **структуру** a₂, но конкретное значение 1/24 следует из **одномерного** Casimir (через ζ_R(-1)), а не из явного 4D вычисления.
