# §1. СПЕКТРАЛЬНАЯ ГЕОМЕТРИЯ L(2,1)

## 1.1 Определения

**Линзовое пространство** $L(p,q) = S^3 / \mathbb{Z}_p$, где $\mathbb{Z}_p$ действует на $S^3 \subset \mathbb{C}^2$:
$$(z_1, z_2) \mapsto (e^{2\pi i/p} z_1, e^{2\pi i q/p} z_2)$$

Для $p=2, q=1$: $L(2,1) \cong \mathbb{RP}^3$ (проективное пространство).

---

## 1.2 Спектр скалярного лапласиана

### Теорема (Ikeda 1979)

На $S^3$ радиуса $R=1$ спектр $\Delta_0$ (скалярный лапласиан):

| $n$ | $\lambda_n$ | $d_n$ (кратность) |
|-----|-------------|-------------------|
| 0 | 0 | 1 |
| 1 | 3 | 4 |
| 2 | 8 | 9 |
| 3 | 15 | 16 |
| $n$ | $n(n+2)$ | $(n+1)^2$ |

**Формула:** $\lambda_n = n(n+2)$, $d_n = (n+1)^2$

### Проекция на RP³

Антиподное отображение $\iota: (z_1, z_2) \mapsto (-z_1, -z_2)$ действует на сферические гармоники:
$$\iota Y_n^{(m)} = (-1)^n Y_n^{(m)}$$

**Следствие:** На $\mathbb{RP}^3 = S^3/\mathbb{Z}_2$ остаются только **чётные** $n$.

Кратность на $L(2,1)$:
$$d_n^{L(2,1)} = \frac{1}{2}(1 + (-1)^n)(n+1)^2 = \begin{cases} (n+1)^2, & n \text{ чётно} \\ 0, & n \text{ нечётно} \end{cases}$$

## 1.3 Дзета-функция скалярного лапласиана на L(2,1)

### Определение
$$\zeta_{L(2,1)}(s) = \sum_{n=1}^{\infty} \frac{d_n^{L(2,1)}}{\lambda_n^s} = \sum_{k=1}^{\infty} \frac{(2k+1)^2}{[2k(2k+2)]^s}$$

(сумма по чётным $n = 2k$)

### Упрощение
$$\zeta_{L(2,1)}(s) = \frac{1}{4^s} \sum_{k=1}^{\infty} \frac{(2k+1)^2}{[k(k+1)]^s}$$

### Асимптотика (Weyl)
При больших $n$: $\lambda_n \sim n^2$, $d_n \sim n^2$

$$\zeta_{L(2,1)}(s) \sim \int_1^\infty \frac{n^2}{n^{2s}} dn = \int_1^\infty n^{2-2s} dn$$

Сходится при $\text{Re}(s) > 3/2$.

---

## 1.4 Производная дзета-функции в нуле

### Теорема (регуляризация)

$$\zeta'(0) = \lim_{s \to 0} \frac{d}{ds} \zeta(s)$$

связана с функциональным детерминантом:
$$\log \det(\Delta) = -\zeta'(0)$$

### Вычисление для L(2,1)

Используем разложение heat kernel:
$$\text{Tr}(e^{-t\Delta}) = \sum_n d_n e^{-t\lambda_n} \sim \frac{\text{Vol}(M)}{(4\pi t)^{3/2}} + a_1 t^{-1/2} + a_2 t^{1/2} + \ldots$$

Коэффициенты $a_k$ — спектральные инварианты (Gilkey 1975).

**ВАЖНО: Все формулы ниже при $R = 1$ (планковские единицы).**

Для линзового пространства L(2,1) (d=3, R=1):
$$a_0 = \hat{V}_3 = \frac{\text{Vol}(L(2,1))}{R^3}\bigg|_{R=1} = \pi^2$$
$$a_1 = \frac{1}{6}\int_M R\, \frac{d\text{vol}}{R^3} = \frac{1}{6} \cdot 6 \cdot \pi^2 = \pi^2$$ (скалярная кривизна $R=6$)

**Внимание:** Нумерация коэффициентов Seeley-DeWitt:
$$K(t) \sim (4\pi t)^{-d/2} \sum_{k=0}^{\infty} a_{k/2} \cdot t^{k/2}$$

Коэффициенты $a_k$ — **безразмерные** числа при $R = 1$.

Для **3-многообразия без границы** (d=3):
- $a_0 = \hat{V}_3$ — безразмерный объём
- $a_{1/2} = 0$ (нет границы)
- $a_1 = \frac{1}{6}\int_M R\, d\hat{V}$ — первая поправка
- $a_{3/2}$ содержит $\nabla R$ и граничные члены

Для **4-многообразия** $M^4 = L(2,1) \times S^1$ (R=1):
- $a_0 = \hat{V}_4 = 2\pi^3$ (безразмерный объём)
- $a_1 = \frac{1}{6}\int_{M^4} R\, d\hat{V}$
- $a_2 = \frac{1}{180}\int_{M^4} (c_1 R^2 + c_2 R_{\mu\nu}^2 + c_3 R_{\mu\nu\rho\sigma}^2)\, d\hat{V}$

где $c_1, c_2, c_3$ зависят от типа поля, и $d\hat{V} = d\text{vol}/R^4$ — безразмерный элемент объёма.

## 1.5 Формула для ζ'(0) через heat kernel

$$\zeta'(0) = -\gamma a_0^{(D)} + \text{FP}\int_0^\infty t^{-1} [\text{Tr}(e^{-t\Delta}) - a_0 t^{-D/2} - \ldots] dt$$

где FP — конечная часть (finite part), $\gamma$ — постоянная Эйлера.

### Численный результат (mpmath)

```python
from mpmath import mp, nsum, diff, log, pi, inf
mp.dps = 50

def zeta_L21(s):
    """Zeta function for scalars on L(2,1)"""
    def term(k):
        n = 2*k  # even n
        d = (n+1)**2
        lam = n*(n+2)
        return d / lam**s
    return nsum(term, [1, inf])

# Производная в s=0
zeta_prime_0 = diff(zeta_L21, 0)
print(f"ζ'_L(2,1)(0) = {zeta_prime_0}")
```

## 1.6 Связь с 1/24

### Casimir energy на окружности

Для свободного скалярного поля на $S^1$ периода $L$:
$$E_{Casimir} = -\frac{\pi}{6L} \cdot \frac{1}{24} = -\frac{\pi}{144 L}$$

Коэффициент $1/24$ — это $\zeta_R(-1)$ (дзета Римана):
$$\zeta_R(-1) = 1 + 2 + 3 + \ldots = -\frac{1}{12}$$

и $-\frac{1}{2}\zeta_R(-1) = \frac{1}{24}$.

### Обобщение на L(2,1)

**Гипотеза:** Для L(2,1) коэффициент Casimir имеет вид:
$$\kappa_{L(2,1)} = 1 \cdot \frac{1}{24} + O(\text{topology})$$

где единица — это "эффективное число измерений" в IR-пределе.

## 1.7 Таблица спектров L(2,1)

| Поле | $\lambda_n$ | $d_n$ на $S^3$ | $d_n$ на $L(2,1)$ | Чётность |
|------|-------------|----------------|-------------------|----------|
| Скаляр | $n(n+2)$ | $(n+1)^2$ | $(n+1)^2$ (чёт.) | even n |
| Вектор | $(n+1)^2$ | $2n(n+2)$ | $2n(n+2)$ (чёт.) | even n |
| Спинор | $(n+3/2)^2$ | $2(n+1)(n+2)$ | $2(n+1)(n+2)$ (нечёт.) | odd n |

### 1.8 Spin-структуры на L(2,1)

**Теорема (Bar 1992):** L(2,1) = RP³ допускает ровно **две** неэквивалентные spin-структуры.

Spin-структура определяется классом $\eta \in H^1(M; \mathbb{Z}_2) \cong \mathbb{Z}_2$.

| Spin-структура | η | Спектр Дирака |
|----------------|---|---------------|
| Тривиальная | 0 | $\pm(n+3/2)$, n нечётные |
| Нетривиальная | 1 | $\pm(n+3/2)$, n чётные |

**Выбор RPFT:** Используется **тривиальная** spin-структура (η = 0), при которой фермионы "видят" нечётные моды — антиподально-антисимметричные.

**Независимый принцип выбора (bounding spin structure):** фиксируем spin-структуру на $\mathbb{RP}^3$ как ту, которая **продолжается на естественный 4D spin-заполнитель** $W$ с $\partial W=\mathbb{RP}^3$ (например, диск-бандл над $S^2$ с эйлеровым числом $e=2$, эквивалентно — асимптотическая граница ALE-пространства типа Eguchi–Hanson). Такая постановка является стандартной при определении фермионного детерминанта/фазы через APS-индекс и автоматически убирает произвольность выбора spin-структуры.

В рамках нотации таблицы выше эта продолжаемая (bounding) структура совпадает с выбранной в RPFT (η = 0) и даёт спектр с нечётными $n$.

*Ссылка:* Bär C., "The Dirac operator on space forms of positive curvature", J. Math. Soc. Japan **48** (1996) 69–83.
